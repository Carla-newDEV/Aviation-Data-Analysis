Projeto voltado para análise de dados com o objetivo de gerar estatísticas aplicáveis a situações reais. O trabalho envolve coleta, organização e análise de dados para identificar padrões e apoiar a tomada de decisão.
A etapa de busca e organização dos dados foi liderada por Carla-newDEV.

Etapas:

Coleta de dados
Limpeza e organização
Análise estatística
Geração de insights

Funcionalidades do projeto:

1- Consulta com série temporal e ranking Top-N usando funções de janela.
2- View analítica para identificar registros suspeitos e priorizar ocorrências.
3- Trigger que garante regras de negócio e registra auditoria em log.
4- Stored procedure parametrizada para geração de relatórios.
5- Views para perfis diferentes: gerencial (resumido) e analítico (detalhado).
